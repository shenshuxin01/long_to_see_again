# nodeJS基础
如果你是一个前端程序员，你不懂得像 PHP、Python 或 Ruby 等动态编程语言，然后你想创建自己的服务，那么 Node.js 是一个非常好的选择。

- [nodejs菜鸟教程](https://www.runoob.com/nodejs/nodejs-tutorial.html)

## 运行nodejs代码
```sh
# 检查nodejs是否安装成功
node -v

# 运行x.js文件
node ./x.js

```

# 常用npm命令
npm是一个依赖包包管理器，每个包都是js文件模块的。就相当于maven和java的关系
```shell
# 查看本地安装的所有npm包
npm list -g --depth 0

# 删除本地包
npm uninstall -g xxx

```
